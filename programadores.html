El programador front-end

¿Qué hace?

El front end de un sitio web es la parte visible con la que el usuario interactúa directamente. Así pues, la función del programador es la de convertir el proyecto existente en un sitio web que funcione.

El desarrollador front-end desarrolla los elementos de un sitio web con los que podemos interactuar, y trabaja tanto a nivel de diseño como de desarrollo. Los desarrolladores front-end se encargan de diseñar y desarrollar todos los elementos “visibles” de un sitio web.
